# Pageshop is page factory
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080

    # dev ( default with page index: --page=index )
    npm run dev

    # dev with all pages
    npm run dev --page=.all

    # dev with certain page[s]
    npm run dev --page=index[,page1,page2]

# build for production with minification

    # build ( default with page index: --page=index )
    npm run build

    # build with all pages
    npm run build --page=.all

    # build with with certain page[s]
    npm run build --page=index[,page1,page2]

```
